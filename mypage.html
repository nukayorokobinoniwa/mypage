技術

現在扱える技術を程度を含めて記述していく

  • HTML/CSS

    このページを作ることができるレベル。今後何かしら日記のようなものでも作ってみて技術力を鍛えようかなと

  • ruby

    単純な構造のプログラミングなら作成可能。ただし記述方法を調べる必要はある。

  • C

    細かいライブラリを利用した記述はできないが、ある程度の記述なら可能

  • C#

    オブジェクト指向を理解していないため簡単な記述なら可能。今後精進していきます。

  • JavaScript

    ネットの記事を見ながら本当に簡易的な文章なら記述可能。clusterのJavaScriptを書くことが多い。

  • Unity

    2D、3D両方使用可能。VRは未学習のためできない。CCKを利用してclusterのワールドは作ることができる。 基本的なコンポーネントは扱える。アニメーションとアニメーターに関しては勉強中、3Dモデルの利用は勉強中。 猫本にて勉強を開始し、猫本の中身は一通り扱える。

勉強内容

今後勉強していく内容について記述する。

  • HTML/CSS

    細かい記述方法やmeta情報等を理解する。より良いデザイン等ができるように努力する。 そのためにも日記のようなものを毎日記述し慣れようと思い始めた。

  • Git/GitHub

    VScodeの拡張機能を使ったpushのみできる現状だが、しっかりコマンドでもできるようにしつつ、ブランチを生やしたりissueを解決したりと様々な方法でgithubに慣れていけたらと思っている。

  • C#

    オブジェクト指向を理解し、名前空間等を正しく扱えるようになり、継承させつつきれいなコードを書けるようにしていきたい。

  • Unity

    VRをいじれるようになりたい。またアニメーションをしっかり理解してアニメーターをスクリプトを用いていじれるようになりたい。 また3Dモデルを自由自在に操れるようになりたい。unityに関してはとがった人類になることを目標に頑張りたい。
